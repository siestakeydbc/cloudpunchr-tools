# SOC Training Loop Helper

This script creates a timestamped Markdown note for each SOC / blue-team lab so you can combine:
- Raw evidence and logs
- AI assistant output (ChatGPT, Gemini, Claude, etc.)
- Your own detections, hypotheses, and lessons learned

## Usage

```bash
cd ~/soc-training-loop
./soc_training_loop.sh
